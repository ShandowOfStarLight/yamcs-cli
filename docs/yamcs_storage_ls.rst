yamcs storage ls
================

.. program:: yamcs storage ls

**NAME**

    yamcs storage ls - List buckets or objects


**SYNOPSIS**

    ``yamcs storage ls [-l] [-r, -R] [BUCKET]``


**POSITIONAL ARGUMENTS**

    .. option:: [BUCKET]

        Bucket or object.


**OPTIONS**

    .. option:: -l

        List in long format.

    .. option:: -r, -R

        List recursively.
